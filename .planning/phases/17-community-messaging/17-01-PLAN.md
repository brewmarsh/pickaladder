<task type="auto">
  <name>Task 2: Define Firestore Security Rules</name>
  <files>firestore.rules</files>
  <action>
    Create a `firestore.rules` file in the project root to secure the messaging system.
    Rules should enforce:
    1. Users must be authenticated to read or write any data.
    2. Conversations: Users can only read/list conversations where they are a participant (`request.auth.uid in resource.data.participants`).
    3. Messages: Users can only read/write messages in a conversation if they are a participant of the parent conversation.
    4. Participants array in conversations should be immutable after creation.
  </action>
  <verify>
    <automated>test -f firestore.rules</automated>
  </verify>
