  <files>pickaladder/templates/layout.html, pickaladder/static/js/chat.js</files>

    1. Update `pickaladder/templates/layout.html` to include the Firebase Firestore SDK script: `https://www.gstatic.com/firebasejs/8.10.1/firebase-firestore.js`.
    2. Create `pickaladder/static/js/chat.js`. This script should:
       - Initialize a Firestore listener (`onSnapshot`) for the current conversation's messages collection.
       - Render new messages dynamically in the DOM.
       - Implement auto-scroll to the bottom when new messages are added.
       - Handle optimistic UI: append the sent message immediately with a 'pending' state if possible, or just rely on the fast Firestore update.
       - Use `firebase.firestore().collection('conversations').doc(cid).collection('messages').orderBy('timestamp', 'asc')`.
  </action>

  </verify>
